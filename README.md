# douyinquick
一款抖音解析工具app
1、权限：悬浮窗、联网、访问储存
2、使用：抖音app（国内版）；打开软件，给予权限（重启app），自动跳转到抖音，找到自己喜欢的视频->分享->复制链接,之后按提示操作。视频下载在DouyinQuick目录下。

19-06-04:接口变更，修复完成  
19-06-05:支持长视频(完整版)解析  
19-07-16:优化界面  
19-08-05：源代码里面的接口不再使用，如需使用本APP请下载新版[DouQuick-v1.5.apk](https://lyfzn.top/plugins/BO/tools/mysource/source/DouQuick-v1.5_sign.apk),想自己改的在MyService.java更换API（API在依附项目douyin-clear=php,自行下载部署），置old=1&isFormat=1即可使用;  
19-8-17：修复无水印解析接口，已更新至v1.5，v1.3,v1.4版本将不能解析完整视频，请下载v1.5使用  

优点：  
相较其他解析软件，需要去复制粘贴分享链接才能下载；本APP直接在应用内复制链接即可解析下载（APP运行期间，当你复制来自其他渠道的分享链接时也可触发下载），使用更加便利。
